Quick Start
1. Switch to Watched Dirs and click + Add Folder to choose directories to monitor.
2. Edit any supported file (.md, .txt, .docx, .pdf, .csv, .json, .html, .rtf) in a watched folder.
3. Watch it appear in the Recent tab within seconds — every change is versioned automatically.
4. Open the Web App to see synced artifacts, collaborate with your team, and manage AI agents.
How It Works
The daemon watches your folders using macOS FSEvents. When a file changes, it computes a SHA-256 hash, generates a diff against the previous version, and stores a versioned commit in a local SQLite database at ~/.lurk/ledger.db. Changes are synced to the cloud every 60 seconds.
Key Concepts
Artifact — A tracked file. Each file gets a unique ID and full version history.
Commit — A versioned snapshot. Every change creates a commit with content + diff.
Voice Profile — Your writing style model. The daemon extracts samples to train your Digital Twin.
YOLO Mode — Progressive autonomy where AI agents earn trust and act independently.
